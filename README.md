# 💱 Currency Converter

A simple and user-friendly Currency Converter web application built using **HTML, CSS, and JavaScript**. It fetches real-time exchange rates using a public API.

---

## 🚀 Features

* 🌍 Convert between multiple currencies
* 🔄 Real-time exchange rates using API
* 🇺🇸 Country flags for better UI
* 📱 Responsive design
* ⚡ Fast and easy to use

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API

---

## 🔗 API Used

```
https://latest.currency-api.pages.dev/v1/currencies
```

## 💻 How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/Currency_Converter.git
```

2. Open the folder
3. Run `index.html` in browser
